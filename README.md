# 🚀 Modern Glassmorphism Navbar

A **modern, responsive navbar** built using **HTML, CSS, and JavaScript** featuring:

✨ Glassmorphism UI
✨ Animated gradient border
✨ Smooth underline hover effects
✨ Mobile menu animation

## 🎯 How It Works

* Flexbox is used for layout alignment
* Glass effect created using `backdrop-filter`
* Gradient border uses CSS animations
* JavaScript toggles mobile menu
